# 广角镜头网球场相机相对位姿调研计划

## 目标

调研能够从单目广角/鱼眼视频中估计“相机相对于网球场”的方案，重点解决：

- 光照、场地颜色和遮挡变化造成的传统视觉泛化差；
- 单帧只看到部分球场，关键点不完整；
- 需要输出相机位姿，而不只是把球场平面映射到图像的单应矩阵；
- 广角镜头引入径向/鱼眼畸变。

## 调研路线

1. 检索网球场专用的 GitHub 实现、预训练权重和数据集。
2. 检索体育场地注册、相机标定、部分场地可见、广角/鱼眼模型相关论文。
3. 对每个候选方案记录输入假设、输出、是否支持部分可见、是否支持畸变、代码与许可证状态。
4. 给出适合本项目的候选技术路线和最小验证实验，不直接假设传统线段聚类可以解决全部问题。

## 评估标准

- **场景匹配**：网球场、固定/半固定相机、单目、广角、室外光照变化。
- **部分可见鲁棒性**：少量关键点或局部线段是否仍可注册，是否能利用跨帧信息。
- **几何输出**：能否得到球场坐标系下的相机位置、朝向和高度，而非只有平面单应性。
- **工程可用性**：代码是否可运行、权重/数据是否可获得、依赖和许可证是否清晰。
- **验证成本**：是否能用项目自采视频和少量标注快速验证。
